My top 5 favorite Free and Open Source Video Games

2022-05-18 20:16:12 +0500 PKT Technology Linux Raspberry Pi Gaming

First and for most for disclaimer this list is mine and these are the top 5 FOSS Linux games I like.

0 A.D.

Of course, the first one on my list is 0AD. It is one of the best open-source games out there. It started its life as a mod for Age of Empires but has since become its own thing and runs on its own Pyrogenesis game engine. It focuses on the era between 500 BC — 1 BC.

You can download 0 A.D. from here https://play0ad.com/

Open RA

Open RA is a game engine recreation of the classic Red Alert, Dune 2000 and Red Alert Tiberian Dawn games. It is pretty much the same fun as expected from those classic games but enhanced for the modern systems. The game engine is open-source, but it does require original game data though thankfully it automatically downloads the game data from their freeware versions automatically if nothing provided.

You can download OpenRA from here https://www.openra.net/

SuperTux Kart

SuperTux Kart is a Kart racing game inspired by the classic Super Mario Kart but with free and open-source software mascots. It’s an extremely addictive and fun game and is a thing of its own. It also has several other game modes including a one called Soccer which is a mix between a kart racing game and Rocket League.

The game could be downloaded from here https://supertuxkart.net/

C-Dogs

C-Dogs is one of those games which had existed for a long time since the days of MS-DOS and I have only discovered it recently. It kind of behaves like modern MOBA games. It is a shoot ’em up video game in which you have to kill your enemies or each other in a deathmatch which could single player or it could be cooperative.

You can get more information about downloading the game from here https://github.com/cxong/cdogs-sdl

VCMI

VCMI is a game engine recreation of the classic Heroes of Might and Magic III from 1999. The game is a turn-based strategy game but on steroids set in the Might and Magic world. The game is so deep that you got to play it to believe it. The game is still very popular and has a cult following. VCMI needs original game data but if that is not available VCMI use game data from the shareware version of Heroes of Might and Magic III.

You can download VCMI project from https://vcmi.eu/
