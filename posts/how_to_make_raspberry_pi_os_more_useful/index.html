How to make Raspberry Pi OS Bullseye more useful

2022-03-22 17:48:07 +0500 PKT Technology Linux Raspberry Pi

Raspberry Pi OS is great for beginners but after a while you will start to realize that its desktop environment which is a modification of LXDE environment is rather limited to be used as a desktop. To make Raspberry Pi OS into the more standard Debian distro on which it is based on follow the following steps.

  1. Either select “Raspberry Pi OS Lite (64-bit)" or “Raspberry Pi OS Lite (32-bit)" and write it on your SD Card or USB Drive. If you are using Raspberry Pi 4 or 400 then I will prefer you to install the 64-bit version.

  1. After writing the SD Card or USB Drive boot your Raspberry Pi with it and login with “pi” and give the password “raspberry” then change the password with the command “passwd” to whatever you like.

  2. Then type “sudo raspi-config” and set your localizations and connect to your wifi network if required.

  3. Then type “sudo apt update && sudo upgrade” to update the system.

  4. Now if you want to install like XFCE desktop environment then type “sudo apt install task-xfce-desktop” similarly you can install Gnome and KDE with “sudo apt install task-gnome-deskop” and “sudo apt install task-kde-desktop” respectively. The meta package will take care of mostly everything.

  5. It will take some time to install the desktop environment and all its dependencies. Onces it is done then type “sudo apt purge openresolv dhcpcd5” to remove openresolv and dhcpcd5 otherwise you will not be able to connect to your wifi or ethernet connection via network manager and you have to rely on raspi-config utility. Once both programs are removed then reboot.

Feel free to fine tune it like in the screenshot above and change the theme and the wallpaper as you wish. Type “sudo apt install neofech” to install neofetch and type “neofetch” in the terminal to get the distro information like in the screenshot.

  1. To make Raspberry Pi OS do autologin on your console type “sudo nano /etc/lightdm/lightdm.conf” and under "[Seat:*]" section locate the lines "# autologin-user=" and "#autologin-user-timeout=0” comment both and add the user “pi” after “autologin-user=" as following.

Save the file and reboot the system and now your system will automatically login to the “pi” user.

  1. If you are annoyed by user authentication whenever your computer comes from sleep in XFCE desktop go settings and then power management and disable the “Lock screen when system is going to sleep” in the Security tab.

  1. Now if you are running the Raspberry Pi OS 64-bit version then in the terminal window type “sudo apt install flatpak” and once it is done type “flatpak remote-add –if-not-exists flathub https://flathub.org/repo/flathub.flatpakrepo" to install flatpak on your computer. Once everything is done reboot if necessary. Of course not everything will work but it would still be useful to install many missing packages from the main repo.

  2. With “task-*-deskop”" package by default Raspberry Pi OS will install the Firefox ESR version by default. To install the Raspberry Pi OS modded Chromium on your terminal windows type “sudo apt install rpi-chromium-mods”.

  1. To install the other missing default Raspberry Pi OS packages you can always go to synaptic and search for “raspberry” and install whatever you need. Though beware do not install “raspberrypi-net-mods” at any cost otherwise it will mess up with the Network Manager and you will have to use the raspi-config utility to connect to the internet.

I hope I have made everything easy. If you like my tutorial feel free to share it.
